# TextUtils

TextUtils is a simple and powerful text manipulation and analysis tool. This web application allows users to convert text between upper case and lower case, count words and characters, and estimate the average reading time. It's a handy utility for writers, students, and professionals who need quick and easy text processing.

## Features

- **Text Conversion**: Convert your text to upper case or lower case with a single click.
- **Word and Character Count**: Instantly get the word and character count of your input text.
- **Average Reading Time**: Estimate the average reading time for your text, providing valuable insights for content creators.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (v6 or higher) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/ShivamTiwari0214/textutils.git
   ```
2. Navigate to the project directory:
   ```sh
   cd textutils
   ```
3. Install the dependencies:
   ```sh
   npm install
   # or
   yarn install
   ```

### Running the Application

To start the development server, run:
```sh
npm start
# or
yarn start
```
The application will open in your default web browser at `http://localhost:3000`.

### Building for Production

To create a production build, run:
```sh
npm run build
# or
yarn build
```
The optimized build will be output to the `build` directory.

## Usage

1. Open the TextUtils app in your web browser.
2. Enter your text into the text area.
3. Use the provided buttons to convert text to upper case or lower case.
4. View the word count, character count, and estimated reading time below the text area.

## About

TextUtils was created to provide a straightforward and user-friendly interface for basic text transformations and analytics. The inspiration for this project came from the need for a simple tool that can help users process text quickly and efficiently without the need for complex software.
